# Telegram Setoran Video Bot

Bot Telegram untuk menerima setoran video dan mencatat point otomatis ke Google Sheets.

## Fitur
- Deteksi video + caption
- Caption = point
- Username diambil otomatis
- Input point ke bawah sesuai username
- Validasi username di sheet
- Reply otomatis di grup

## Struktur Sheet
- Row 1 = Username
- Kolom per user
- Point masuk ke bawah

## Deploy
Bot siap deploy di Render.com
